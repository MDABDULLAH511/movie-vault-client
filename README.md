**Movie Vault - Turn your movie collection into a vault.**

**Purpose**
The purpose of Movie Vault is to provide users with an easy and efficient way to manage their movie collections. This web application allows users to store, organize, and explore movies in one secure digital space. Built with React, Firebase, and MongoDB, the project demonstrates modern web development concepts such as CRUD operations, authentication, routing, and responsive design. The main goal of this application is to enhance user experience through an intuitive interface while showcasing practical implementation of full-stack development skills.
